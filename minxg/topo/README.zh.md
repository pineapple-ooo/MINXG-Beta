# 代数拓扑

> MINXG 6 大数学支柱第 4 个,53 个算子,ID 8000-8052。

**拓扑数据分析(TDA)**原语作为一等公民算子。持续同调(H₀、H₁、H₂)、Betti 数、持续图、Wasserstein 距离、Mapper 算法 —— 全部纯 Python。

## 为什么对 AI 重要

1. 损失景观有拓扑结构
2. 嵌入流形有非平凡拓扑
3. 持续特征是尺度不变的
4. Wasserstein 距离是拓扑特征上的恰当度量
